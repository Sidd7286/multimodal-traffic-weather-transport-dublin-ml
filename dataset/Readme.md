## Data Sources

Due to GitHub file size limits, raw datasets are not stored in this repository.

The project uses the following publicly available datasets:

1. **SCATS Traffic Flow Data (Dublin)**
   - Source: Dublin City Council / Smart Dublin
   - Link: https://data.smartdublin.ie/
   - Description: Traffic volume, congestion and saturation data for Dublin junctions.

2. **Weather Data (DLY532 – Met Éireann)**
   - Source: Met Éireann
   - Link: https://www.met.ie/climate/available-data
   - Description: Daily weather observations including temperature, rainfall and wind speed.

3. **GTFS Realtime Trip Updates**
   - Source: National Transport Authority (NTA)
   - Link: https://api.nationaltransport.ie/
   - Description: Real-time bus delay and trip update data.

4. **GTFS Text Alerts / Delay Messages**
   - Source: Transport for Ireland (TFI)
   - Link: https://www.transportforireland.ie/open-data/
   - Description: Text-based service alerts and delay messages for NLP analysis.
